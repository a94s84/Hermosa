以服飾電商為主題的 SPA 網站。

商品文案、圖片來源 : [mouggan](https://www.mouggan.com/zh-tw/)

---
### 使用技術
- 以 `Vue 3` 與 `Vue Cli` 架設開發。
- 前台手刻 CSS，後台使用 `Bootstrap 5` 框架，以進行切版。
- 加入 `ESLint standard` 統一程式碼結構。
- 使用 `Pinia.js` 傳遞跨元件資料，作為狀態管理工具。

### 使用套件
- `Vue 3`
- `Vue Cli`
- `Vue Router`
- `Vue Axios`
- `vue-loading-overlay`
- `Vee Validation`
- `pinia`
- `Bootstrap 5`
- `lodash`
- `Swiper`
